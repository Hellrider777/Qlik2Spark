# Qlick2Spark

A tool for converting Qlik scripts to Apache Spark code.

## Features

- Convert Qlik Sense/QlikView scripts to Spark SQL
- Support for common Qlik functions and expressions
- User-friendly web interface

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd qlik2spark
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

1. Upload your Qlik script file
2. Review the converted Spark code
3. Download or copy the generated code
